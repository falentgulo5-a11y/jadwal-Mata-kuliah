# Jadwal Matkul

Project ini berisi jadwal mata kuliah harian secara sederhana tanpa database. Semua data disimpan langsung di kode.

## Jadwal Matkul Mingguan

| Hari    | Mata Kuliah                   | Jam       |
|---------|-------------------------------|-----------|
| Senin   | Matematika Diskrit            | 08:00-10:00 |
| Selasa  | Pemrograman Dasar             | 10:00-12:00 |
| Rabu    | Basis Data                    | 08:00-10:00 |
| Kamis   | Algoritma & Struktur Data     | 10:00-12:00 |
| Jumat   | Sistem Operasi                | 08:00-10:00 |
| Sabtu   | Praktikum Pemrograman         | 10:00-12:00 |
| Minggu  | Libur / Review Materi         | -         |

## Tujuan
- Membantu mahasiswa mengatur jadwal kuliah
- Mempermudah melihat mata kuliah dan jamnya
